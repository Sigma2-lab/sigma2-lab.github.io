MVP

Model Logic

How the allocation is derived:

  • Input: User-defined macro regime (Text-to-Market constraints).
  • Process: LLM translates regime into relative view vectors (High/Medium/Low).
  • Optimization: Python backend runs Mean-Variance Optimization (MVO) subject to regime constraints.
  • Risk: Covariance matrix estimated using regime lookback on ETF proxies.

Connection

Your API key is sent directly to our server to make a single Anthropic request and is never stored or logged.

First time? Quick setup
  1. The Backend URL is pre-filled — leave it unless you're running your own instance.
  2. Get an Anthropic API key from console.anthropic.com/settings/keys and paste it below. It never leaves your browser except as part of each API request.
  3. Describe a macro regime in the chat (e.g. "soft landing with sticky inflation", "stagflation shock") and hit Send. The LLM will propose allocation bands you can adjust before optimizing.
